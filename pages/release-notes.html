Velog Stats Release Notes

Velog Stats 확장 프로그램의 모든 버전 변경 사항을 확인하세요.

버전 1.8.0

주요 변경사항: 로그 관리 기능 추가

  • Chrome Storage 활용
    • 조회한 로그 저장 기능
    • 저장된 로그 삭제 기능

버전 1.7.1

주요 변경사항: 기능 변경 없음, UI 개선

  • Web Store 이미지 변경

버전 1.7

주요 변경사항: 게시물 정렬 기능 추가 및 문구 개선

  • 게시물 정렬 기능
    • 조회수 기준 정렬
    • 댓글 수 기준 정렬
    • 좋아요 수 기준 정렬
  • 문법 상 어색한 문장 수정

버전 1.6

주요 변경사항: Velog 통계 기능 구현, access token 자동 추출로 사용자 경험 개선

  • 게시물 통계 제공
    • 조회수
    • 좋아요 수
    • 댓글 수
  • 모든 게시물에 대한 통계를 한번에 불러오기 기능
  • Velog의 access_token 쿠키 값을 사용한 인증 처리

Demo 버전

주요 기능: Velog ID와 access token을 직접 입력하여 통계 조회

  • 사용자 입력 필드
    • Velog ID 입력란
    • Access Token 입력란
  • 통계 조회 기능
    • 입력된 ID와 token을 사용하여 Velog API 호출
    • 게시물 목록 및 각 게시물의 상세 통계 조회
  • 표시 정보
    • 총 게시물 수
    • 총 조회수
    • 총 좋아요 수
    • 총 댓글 수
    • 게시물 별 상세 통계 (제목, 조회수, 좋아요 수, 댓글 수)
